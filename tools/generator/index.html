Files on this Paper

Each file gets its own QR code. HTML files can reference other files on the same or different papers using absolute tagdrop:// links — the app intercepts these and serves the cached bytes inline, so images, audio, and SVGs load without a network connection:
<img src="tagdrop://ROOTHASH/photo.jpg">
<audio controls src="tagdrop://ROOTHASH/song.mid">
Slugs can contain / to mimic folder paths (e.g. images/photo.jpg) — they are matched as plain strings.
